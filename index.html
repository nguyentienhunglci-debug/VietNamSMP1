Lệnh đăng kí

Để tạo tài khoản, bạn vào server và gõ lệnh sau:

/register <mật khẩu> <nhập lại mật khẩu>

Ví dụ: /register 123456 123456

Lệnh đăng nhập

Để đăng nhập vào tài khoản đã tạo, bạn gõ lệnh sau:

/login <mật khẩu>

Ví dụ: /login 123456

Lệnh đổi mật khẩu

Để thay đổi mật khẩu, bạn gõ lệnh sau:

/changepassword <mật khẩu cũ> <mật khẩu mới>

Ví dụ: /changepassword 123456 654321
